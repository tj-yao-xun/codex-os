---
name: surgical-bugfix
description: Minimal-diff bugfix workflow for existing codebases.
---

When using this skill:

1. Restate the bug in one sentence.
2. Identify the smallest likely set of files involved.
3. Reproduce the issue or inspect the exact failing path.
4. Make the minimum viable fix.
5. Avoid refactoring adjacent code unless required.
6. Run focused verification.
7. Report:
   - root cause
   - files changed
   - verification performed
   - residual risk
