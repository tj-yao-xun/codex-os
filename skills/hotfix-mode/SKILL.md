---
name: hotfix-mode
description: Emergency fix workflow optimized for restoring correctness quickly while documenting compromises.
---

When using this skill:

1. State clearly that this is a hotfix.
2. Prioritize restoring service or correctness first.
3. Minimize blast radius.
4. Clearly label all temporary compromises.
5. Suggest proper follow-up remediation work.

Rules:
- Do not silently leave emergency hacks undocumented.
- Keep the hotfix narrowly scoped.
- Separate the urgent fix from the later proper solution.
